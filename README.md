# 欢迎加入冬季战场！
---

### 一、项目简介

**冬季战场（Winter Battlefield）** 是一个基于《我的世界》Java版 1.21.11 的多人对抗数据包。它不是小游戏插件，不是命令方块堆砌的临时娱乐——它是一个从底层重构、纯函数驱动、支持多语言与主题包的**完整对战平台**。

**核心理念**：免费、公平、文化传播、玩家共创。

- **免费**：永远不收一分钱，如果谁让你付费购买，你被骗了。
- **公平**：禁止外挂，不搞氪金变强，每一场胜负只取决于你的操作和配合。
- **文化传播**：从“雪域龙腾”到“极地潜兵”，冬战可以被任何人改写成任何文化主题。
- **玩家共创**：每一个测试员、每一个提交翻译键的人、每一个在60144发过“Ciallo～”的人，都是冬战的共建者。

**一句话定义**：  
**在冰原上打一场15分钟的攻防战，用兵种、导弹和步话机，写自己的战争故事。**

### 二、快速开始

#### 1. 安装
- **数据包**：将 `djzc444.zip` 放入存档的 `datapacks` 文件夹。
- **资源包**：将 `赛扬的冬季战场v1.0.6.zip` 放入 `resourcepacks` 文件夹，并在游戏中加载（**优先级调到最高**）。
- **可选主题包**：如需“雪域龙腾”或“极地潜兵”，同样放入资源包文件夹，优先级高于默认包。

> 📌 提示：强烈推荐使用服务器进行多人对战。虽然单人模式也可体验游戏的部分内容，但乐趣在于对抗。

#### 2. 进入游戏
- 创建/进入世界后，你会自动获得 **冬战成员用书**（一本成书）。
- **按 G 键**（或右键成员用书）打开主菜单。

#### 3. 第一次对战流程
1. **选择阵营**：CT（防守方）或 T（进攻方）。  
   - CT：阻止T占领全部战区，拖到15分钟结束即赢。  
   - T：占领全部四个战区（A、B、C、D）即赢。
2. **选择兵种**：从四大类（突击、侦察、支援、航空）8个亚种中选一个。新手推荐“支援兵”或“近战兵”。
3. **点击“准备”**：等待所有玩家准备，游戏自动开始。
4. **传送**：在成员用书中打开“传送系统”，选择战区、启动器或队友身边出发。
5. **开战**：使用步话机（G键菜单内）发送指令，配合队友占点、激活启动器、发射导弹。

#### 4. 基础操作速记
- **G键**：主菜单（兵种、传送、步话机、百科）
- **右键成员用书**：同上
- **步话机指令**：在主菜单中点击预设消息（如“进攻A点！”“发现敌人！”）
- **导弹发射**：手持导弹（如“空空导弹”），右键发射。
- **干扰导弹**：按 Q 键投掷任意物品（鸡蛋、雪球等），导弹会追踪干扰物。

> ⚠️ 注意：敌方玩家头顶**不显示名字**，只能通过盔甲纹饰颜色（CT蓝色 / T红色）和外观识别。

### 三、游戏机制详解

#### 1. 战区与占领
- **四个战区**：A（冰湖北侧）、B（冰刺之地南侧）、C（雪原村庄中心）、D（CT家后方冰湖）。
- **占领规则**：
  - T方进入战区 → 增加占领进度；CT方进入 → 减少进度。
  - 离开战区不重置进度（保留已占领比例）。
  - 双方同时进点时，T方进度增加效率减半，CT方扣除效率不变。
  - CT无法夺回已被T完全占领的点位。
- **占点时间**：从30秒调整为60秒（1.0.6二测后），让防守方有回防空间。

#### 2. 启动器（C/D点前置条件）
- **C点启动器**（C1、C2、C3）：分别位于南侧战壕地堡、村庄地道、北部山腰。
- **D点启动器**（D1、D2、D3）：分别位于碉堡、灯塔地下室、CT出生点。
- **激活方式**：占领B点后，启动器高亮；玩家需**旋转物品展示框7次**。
- **全部激活后**：对应战区开放占领。
- **自动修理**：如果展示框被意外击毁，系统会自动重新放置（1.0.6新增）。

#### 3. 兵种系统
| 大类 | 亚种 | 定位 |
|------|------|------|
| **突击兵** | 近战兵（盾斧剑）、三叉戟兵（破盾戟） | 正面推进 |
| **侦察兵** | 狙击手（大狙）、游骑兵（高速矛） | 远程压制/骚扰 |
| **支援兵** | 防空兵（霰弹弩+导弹）、冈格尼尔（秒杀三叉戟） | 火力支援/辅助 |
| **航空兵** | 立体机动装置（导弹+雷枪）、板载兵（自杀核弹） | 制空与奇袭 |

- **兵种解锁**：立体机动装置需累计5击杀；板载兵一局仅一次。
- **平衡调整**（1.0.6三测前）：狙击手伤害降至44（略低于近战剑），导弹追踪速度提升至1.65m/tick。

#### 4. 导弹与空战
- **导弹类型**：
  - 空空导弹：追踪敌方空中单位。
  - 空地导弹：追踪地面单位。
- **发射与干扰**：右键发射；任何投掷物（鸡蛋、雪球）会吸引导弹追踪，可用于欺骗。
- **告警系统**：当敌方导弹进入30米内，**每秒发出一次告警**（距离越近越急促）。
- **机场补给**（1.0.6新增）：
  - 己方机场停留30秒 → 获得4枚空空、8枚空地导弹。
  - 接近敌方机场（<75米且有敌方单位）→ 机场自动发射地空导弹。
  - 机场设有 **弹射器**（绊线钩+命令方块），给予速度和漂浮效果，模拟航母弹射。

#### 5. 步话机与战术通信
- **MK.IV 步话机**：基于对话框的预设指令系统。
- **12条指令**：包含“进攻A点”“防守D点”“发现敌人”“我来助你”等。
- **彩蛋指令**：`Ciallo～(∠・ω< )⌒★` —— 彩蛋。
- **使用方式**：G键主菜单 → “转到步话机” → 点击对应消息。

#### 6. 传送系统（第三代）
- **20个传送点**：包括战区、启动器、出生点、机场、队友身边等。
- **条件限制**：
  - 只能传送到己方控制的区域。
  - 地下（Y<40）可随意传送（用于配装中心）。
  - 传送到最近/随机队友（1.0.6新增）。
- **旧版传送器**：保留但已加警示标语，建议使用新版。

#### 7. 特殊机制
- **轰炸塔**：地图中隐藏的观测塔，使用“轰炸指示器”标记区域，5秒后轰炸半径7米（**可能炸死自己**——祖传bug，不修）。
- **自动重生点**（1.0.6新增）：
  - 开局时重生点自动设为地下配装处。
  - 游戏结束时重生点移到队伍选择器附近。
- **防破坏**：所有生存模式玩家进入服务器自动设为冒险模式，无法破坏方块。

大战场玩法本身相当有趣，但是——
数值平衡有待改进
神枪·冈格尼尔左键突进冷却时间太短，很难正常普攻
狙击手太强势了，玩家很容易死掉

有一堆BUG！！！
作为离线服务器，无法正确显示其他玩家的皮肤，也没有加装相应的补丁解决这个问题
赐福系统不平衡，如果不使用**存护**赐福，玩家很容易死掉
……

**在此之外，还有如下问题：**

极少有人使用步话机系统，且QQ语音通话较为混乱，难以有效组织
（有玩家指出可以使用服务端通话插件等方式解决这个问题，我们正在考虑）

没有使用replay mod记录游戏进程
（很难反复检查测试过程，只能靠回忆和日志）

新手引导几乎完全缺失，新人难上手
（比如在寻找加入阵营的按钮和分辨自己所处阵营时比较混乱。因为冬战成员用书因为技术问题没有迁移，这个问题更加突出了。
我们正在制作这样一套系统：
准备系统、阵营选择、兵种选择、赐福选择、出生点选择从地图内集成到对话框中，并使用**成书**作为触发开关，做成新一代的**冬战成员用书**）

存在玩家们把掉刻误认为是网络问题的可能性
（因为只有常驻的几位测试员通过mod发现了服务器掉刻）

中途加入的玩家会加载出生点区块以及它旁边的红石机器
（可能是掉刻的原因之一，我们正在加急制作独立于原有存档的冬战地图，并禁用生物生成等一切无关功能）

有几位中途加入的玩家无视语音沟通，以创造模式误入战场
（我们想到的办法是，将默认游戏模式调节为旁观模式，如果仍有这样的情况，让管理员手动调整他们的游戏模式为旁观模式）

第一梯队：近战兵 狙击手
第二梯队：冈格尼尔干员 
第三梯队：普通兵 三叉戟兵 立体机动装置 板载兵
第四梯队：游骑兵（没有人玩）

#### 冬季战场v1.0.6 第二次测试报告：

服务器的准备工作：
- 服务器设备与第一次测试（alpha_test）相同
- 服务器稳定以20TPS运行，最高MSPT约为22~25

- 极限情况下，有4人参与战斗

根据大家的反馈，整理出以下的观点：
- 狙击专用弹数值远高于近战武器，使用狙击专用弹输出风险低，回报高
- T方攻占点位速度过快，CT回防难

- 游骑兵伤害不如预期
- 尽管有玩家名牌和盔甲纹饰，但除了无法对队友造成伤害以外，难以分辨场上人物的阵营
- 建议通过削弱武器伤害，增加防护等方式延长冬季战场的TTK

在玩家反馈的问题以外，我们发现了如下的问题：

- 即使有系统消息提示，需要旋转物品展示框激活启动器，但新玩家常常会忽视系统提示，导致无法推进游戏

- 在没有说明的情况下，新玩家不会知道按G键展开冬战主菜单

- 三叉戟兵出场率较低

  由于campfire_cosy_smoke粒子的特性，实际的视觉效果大概是粒子从“爆心”出发，因为“冲击波”快速扩散，爆心中央升起一条细细的烟柱；因为该粒子会被实体方块阻挡，粒子会被墙壁挡住，地面上会留下一些看起来像“灰烬”的效果，在80到129游戏刻后消失
- 现在，当debug模式开启时，会解除轰炸指示器的激活范围限制和耐久限制
- 现在，轰炸指示器的伤害半径从7米扩大到14米，但伤害从5000固定伤害改为按照相对爆心的距离衰减
- 为轰炸指示器增加了音效
- 为轰炸指示器增加耐久限制：使用两次以后，轰炸指示器被系统没收
- 现在，轰炸指示器的轰炸标点移除机制被移除了，每一个被打出的轰炸标点，使命必达！
- 因为轰炸指示器damage指令逻辑的变更，现在，轰炸指示器无法炸死自己了（但是依旧可以炸死队友）

- ##### 兵种管理系统
- 增加了[兵种管理系统]，可以通过该系统，控制兵种是否被启用：
    增加了复选框：
    - 被勾选的兵种将转为启用状态，可以通过勾选复选框，调整兵种是否启用
    - 复选框的字体会随着兵种是否启用显示红色（禁用）/绿色（启用）
    - 复选框的勾选状态会随着兵种是否启用而变化
    增加了8种预设：
    - 陆战-经典模式：默认模式，所有兵种处于启用状态
    - 陆战-冈格尼尔：仅启用干员-冈格尼尔，这也是我们最经典的活动模式之一
    - 陆战-鹰击长空：仅启用航空兵种（板载兵+立体机动装置）和防空兵，保留五杀激活航空兵，板载兵只能玩一次的限制
    - 陆战-近距突击：仅启用近战兵和三叉戟兵
    - 陆战-狙击模式：仅启用狙击手
    - 冬战-怀旧模式：仅启用御三家（近战兵、狙击手、普通兵）
    - 陆战-随机兵种：随机抽取启用的兵种（有概率选到全部禁用，建议重新抽一次）
    - 海战-先遣模式：增加了海战入口，预计于1.1版本实装


- ##### 自动设置重生点
- 增加了自动设置重生点功能：
- 游戏开局时，双方玩家的重生点会自动设定为己方地下配装处
- 游戏开局时，双方玩家会被自动传送到己方地下配装处
- 游戏结束时，所有玩家的重生点会自动设定到队伍选择器附近
