### 游戏逻辑优化
将计分板**prepare**的准则从**dummy**改为**trigger**
移除了**bossbar**中，已经被弃用的部分
移除了C点，D点使用的旧版占点系统（离开点位后，占领进度重置，双方同时进点时，占领暂停），改为新版占点系统（离开点位后保留占点进度，双方同时进点时，T阵营仍会增加占领进度，但效率下降一半）
将**步话机**对话框的触发方式从**右键所有成书**改为**右键带有[msg]标签的成书**，以兼容旧版**冬战管理用书**和**冬战成员用书**
游戏结束后，除了原有的**title**提示之外，还增加了**tellraw**提示游戏结束，并通过**click_event**制作了快速进入准备状态的功能

### 数据包结构优化
将**start.mcfunction**从数据包函数根目录移动到**game**文件夹
将**mine.mcfunction**从数据包函数根目录移动到**game**文件夹，重命名为**lib_mine.mcfunction**
将**type_ltjd.mcfunction**的**导弹**部分转移到**missile_aa.mcfunction**和**missile_ag.mcfunction**
将**导弹**系统中的成就检测代码转移到**advancement/detection.mcfunction**处理


### 武器系统改动
**神枪-冈格尼尔**增加了**突进**附魔，现在，【干员·冈格尼尔】可以使用**神枪-冈格尼尔**进行突进
移除了**一键三连**的旧版三连发机制，但依旧保留了**真的可以三连！**作为绝版成就
**导弹系统**中，取消了作为导弹的**烟花火箭实体**的Motion，使其更容易控制；导弹从**时间引信**（导弹在发射后一段时间自动爆炸）改为**近炸引信**（导弹接近敌人时自动爆炸）。现在，导弹会在接近到敌方空军单位3米以内时自动爆炸
因为BUG频发，移除了**使用特殊的三叉戟**触发导弹的方法，**重新启用**鸡蛋和雪球触发导弹的方式
为**雷枪**增加了附魔光效，使其易于辨认

### BUG修复
修复了C点**occupy**计分板检测的空间范围与C点实际位置不一致的BUG
修复了bossbar**minecraft:c**的名字显示为小写的BUG
修复了bossbar**minecraft:d**的名字显示为小写的BUG
修复了因**目标选择器**语法变化，导致**垂直机动系统**失效的BUG
